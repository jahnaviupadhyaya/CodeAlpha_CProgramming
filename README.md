# CodeAlpha_CProgramming
C Programming project for internship in CodeAlpha.

TASK 1: Basic – Calculator Program
A C program to perform basic arithmetic operations (Addition, Subtraction, Multiplication & Division).
Use switch case for operation selection.

TASK 2: Medium – Matrix Operations
A C program to perform matrix addition, multiplication and transpose.
Using function and 2D array.

TASK 3: High- Student Management System
A C program to manage student record features: Add, Delete, Update, Search, Display records.
Using structures and file handling to store data permanently.
